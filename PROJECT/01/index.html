03. Problems & Solution

  • 화면 크기에 따른 gird 레이아웃 오류

    모바일에서 주었던 grid값 때문에 태블릿 버전의 레이아웃이 무너짐.
    미디어쿼리를 이용하여 각 레이아웃에 grid값을 다시 적용하여 각 부분 구현.

  • 숫자 카운팅 효과

    .setInterval() /.clearInterval() 이용하여 작업함. 하지만 숫자마다 올라가는 속도가 다르고 아직 사용에 미숙하여 작동에 버벅거림이 있다. 좀 더 쉽게 구현할 방법을 찾아봐야 함.

  • JavaScript .Toggle() 이용

    모바일 버전일 때 메뉴를 css의 hover을 이용하여 구현함. 새로 배운 .toggle()을bg1bg2bg3bg4 이용하여 모바일 버전에서 클릭시 메뉴가 보일 수 있도록 추가 작업함.

04. Review

디자인부터 퍼블리싱까지 처음으로 제작한 첫 프로젝트입니다. 반응형으로 웹을 처음으로 제작하려다 보니 다양한 부분에서 신경 써야 할 것들이 많았습니다. 미디어 쿼리와 vw 가변 단위 값을 이용하여 사용자별 환경, 디바이스 제약을 해결하기 위해 노력하였습니다.

HTML과 CSS의 적용은 어느 정도 적응이 되어 적절한 태그들을 활용하여 레이아웃을 만드는 것은 조금 더 수월해졌습니다. 처음으로 JavaScript를 배웠고, 이후에 적용하며 변하는 웹사이트를 보면 신기하기도 하고, 뿌듯했습니다.

이 작업을 진행하면서 많은 부분에서 고민을 했고, 모르는 부분들을 구글링을 통해 문제를 해결해나갔습니다. 아직 부족한 점도 많지만, 이번 프로젝트를 진행하면서 어떠한 방향으로 더 노력해나가야 하는지 알게 되었습니다.
